.safetensors file format

a performant file format for ML and AI models

by Charles Iliya Krempeaux

The safetensors file format is a performant file format for machine learning (ML) and artificial intelligence (AI) models.

A big part of the performance of the format is because its contents are meant to be memory mapped (mmap) (rather than parsed or copied). Although, this tends to only work on computers with little-endian numbers.

DType

A safetensors file can contain multiple tensors. Each tensor has an element type. These are (currently) the following defined element types, called dtype in the safetensors file format header:
